Messing Around (October 17 2017)

I'm usually pretty good at writing blog posts, but right now I don't know what to say. I've been working a lot on this website today, just having a good time with it. But there is so much to be done. Well, that's not the problem, because I enjoy doing it. There is just so little time, that's the problem.

What's clear is that this is no way to make a blog post. I don't even know if real developers use Bootstrap CSS. It seems like a good idea, but what are these cards? And how can I customize them if I don't know what all these classes are that I'm just loading into my web page? I'm using Bootstrap because I am following along with this blog post that tells you how to get a web developer job in five months, and one of the things you have to do is watch this YouTube playlist that's a little on the older side now, and the guy says to use Bootstrap because everybody does. Then he shows you how to create components on your own. So why not just do that?

I don't know why. Who cares. It's fun for now.

What really bothers me is all this repeated code that I have. I have three pages: photos, blog, and a home page, and all three of those has to have the navbar code and the script codes. That can't be right. I guess it gets factored out with JavaScript or something? A lot of websites now are just one page. Maybe clicking the navbar interacts with the DOM. At this point I don't know anything about the DOM. Anyway I guess you could set it up so that you click on a navbar option and some elements are hidden and others are shown?

For now, I guess I'll just stick with what I'm supposed to be learning, instead of just getting carried away learning how to make this website through inefficient trial and error. There has to be a balance, though, because on this five month plan there aren't any parts where he tells you to go and try what you're learning out. I'm sure that's expected and all, but I'm on a very tight time-frame and I am doing exactly what he says and no more. Well anyway I have been up to now.

The reason I stopped doing that today and tried to play around with the website is because I was watching Douglas Crockford's Front End Masters lectures and I realized I had already forgotten what I learned by taking Code School's JavaScript Road Trip class, and in fact I didn't remember much of anything I had learned, and worst of all I didn't have anything to show for the fact that I had actually learned those things at one point but some stupid badges from Code School and freeCodeCamp. Nobody cares about those badges, I'm sure.

Well, hopefully they'll come here and see that I am the best copy and paster of Bootstrap CSS ever. Sweet.
